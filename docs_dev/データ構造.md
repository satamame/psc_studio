# データ構造プラン

## 台本 (PlayScript)

- title
- plot
- chars (one to many)
- scenes (one to many)

## キャラクタ (Character)

- sortkey
- name
- desc
- sample
    - 改行区切りのセリフのサンプル

## シーン (Scene)

- sortkey
- headline
- chars (many to many)
    - 出ている人物。セリフの有無は問わない。
- sofar
    - ここまでのあらすじ
- sofar_gen
    - ここまでのあらすじ (自動生成)
- outline
    - このシーンのあらすじ
- outline_gen
    - このシーンのあらすじ (自動生成)
- lines
    - セリフ行とト書き行だけの fountain
- lines_gen
    - セリフ行とト書き行だけの fountain (自動生成)
