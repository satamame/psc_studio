# 画面構成プラン

- ホーム画面 (台本リスト)
- 台本画面
    - サイドバー (シーン一覧)
    - メイン (各シーン)

## ホーム画面

- 台本リスト
- 台本の追加
- 台本の削除
- 台本リストの並べ替え

## 台本画面

- シーンを移動する UI
    - 台本全体の情報も各シーンと同じ階層
- 各シーンの情報
    - 台本全体の場合
        - 登場人物
        - あらすじ
    - 各シーンの場合
        - これまでのあらすじ
            - 自動生成/マニュアル
        - このシーンの概要
            - 自動生成/マニュアル
        - このシーンの台本 (fountain)
            - 自動生成/マニュアル
